# Leonard Daniel Rivas Jr. Portfolio

Professional portfolio website showcasing Full Stack Development and AV Integration expertise.

## Features
- Clinical "Neon Yellow & Gray" aesthetic
- Responsive design for mobile and desktop
- Integrated project gallery
- Community service highlights

## Deployment
This site is designed to be deployed on **Cloudflare Pages** via **GitHub**.

### Instructions:
1. Place your 8 project images in this folder.
2. Ensure filenames match the `src` attributes in `index.html`.
3. Push this folder to a GitHub repository.
4. Connect the repository to Cloudflare Pages.
